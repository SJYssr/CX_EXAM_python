# 该项目用于学习通考试客户端

## 项目介绍
- 该项目用于学习通考试客户端，主要功能包括本地题库导入，本地题库搜题，AI搜题（maybe，精力不够的话就不搞）
- 全部为python编写
- 调用SetWindowDisplayAffinity函数，防止考试客户端的截屏，录屏（大致步骤就是这个，还没实现）
- 界面可以自动隐藏或者手动隐藏（隐藏后可以继续使用）
- 可以有不同的透明度
- 可以设置快捷退出按键
# 以上都是废话，一个都没实现呢
